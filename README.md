# Metro Route & Fare Finder Backend Engine

A core Java backend system that models a city's metro transit network using a **Weighted Graph Data Structure**. It efficiently calculates the shortest route between stations and dynamically determines fares and journey times.

## Features
* **Graph-Based Transit Network:** Models stations as vertices and tracks as weighted edges using custom Adjacency Lists.
* **Shortest Path Routing:** Implements optimization algorithms (like BFS and Dijkstra's) utilizing `PriorityQueue` and `HashMap` for fast traversal.
* **Dynamic Fare Engine:** Automatically evaluates real-time transit fares and journey times based on track length parameters and station counts.

## Tech Stack
* **Language:** Java (Core Java, Collections Framework)
* **Concepts:** Data Structures & Algorithms (Graphs, Min-Heap, Queue)
* **Tools:** VS Code, Git/GitHub
